[한글 입력: 홈페이지]
한 단어씩 입력 한 후, 방향키라 엔터로 밑줄을 벗기면 됩니다.
하지만, 메모장에서 복사+붙여넣기를 하는게 훨씬 빠릅니다.


[한글 입력: 파일]
코드에서 한글 주석이 붙어 있는 경우,
txt 파일로, UTF-8로 저장 해서 보내줍시다.


[Git Hub 홈페이지 메뉴]
Code: 해당 브런치(작업 영역)의 파일을 볼 수 있습니다.
Issue: 쉽게 말해서 대화방입니다.
Pull requests: 작업 내용 제출에 대해 마스터에게 승인을 구하는 곳입니다.
Project: 작업 상황을 알아보는 곳입니다.
Insight: 이 깃에 대한 정보입니다.


[Git 용어: 커밋과 푸시]
Commit: 파일을 임시 등록합니다. 첫번째 Commit 파일은 바로 Push로 넘어갑니다.
Push: 기존에 있던 파일을 덮어 씌어 등록합니다. 그렇다고 해서 막 Push하지 맙시다.


[Git의 기본]
우리가 쓰는 온라인 저장소는 >> https://github.com/Brogi/Altino1.git
git init			   // 현재 폴더를 로컬 저장소로 지정.
git add				   // 파일 추가
git commit -m "메세지"		   // 커밋 "메세지 내용"
git remote add origin 주소.git     // 폴더와 주소(온라인 저장소)로 연결.
git push			   // 푸시
